## Explore Proxy

Small Node.js proxy for the advanced `蜴ｻ謗｢邏｢` flow.

Environment variables:

- `ARK_API_KEY`: Volcengine Ark API key
- `PORT`: local listen port, default `3010`
- `ARK_BASE_URL`: optional override, default `https://ark.cn-beijing.volces.com/api/v3/images/generations`

Run:

```bash
node server.js
```
